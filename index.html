What is Multimedia?

Multimedia is the combination of two or more media types such as text, images, audio, video, and animation delivered through a digital device. It allows creators to communicate ideas in richer, more engaging ways than a single medium can.

What is Authoring?

Authoring in multimedia refers to the process of creating and combining different media elements using specialized software called authoring tools. It involves designing, assembling, and publishing interactive multimedia content.

3D Solar System

A 3D model of the solar system built using Blender, featuring an interactive age calculator that computes how old you are on each planet based on their orbital periods.

P L A N E T S

What is 3D Modeling?

3D modeling is the process of creating a three-dimensional representation of an object using specialized software. In this project, each planet was modeled and textured in Blender to replicate the look of our solar system.

What is Blender?

Blender is a free and open-source 3D creation tool that supports modeling, animation, rendering, and more. It was used to build and animate all the planets in this project with realistic materials and lighting.

The Age Calculator

Each planet takes a different amount of time to orbit the Sun. By entering your age on Earth, the calculator converts it into your equivalent age on every other planet based on their orbital periods.

What did I learn?

Through this project I learned how to navigate Blender, apply textures and lighting to 3D objects, and combine interactive elements with visual design to create an engaging multimedia experience.

1-Min Rotoscope

A one minute animated video produced in Adobe After Effects using the rotoscoping technique, blending live footage with frame-by-frame illustration.

What is Rotoscoping?

Rotoscoping is an animation technique where animators trace over live action footage frame by frame to produce realistic movement. The result is a hand drawn animation that follows the natural motion of real subjects.

What is Adobe After Effects?

Adobe After Effects is a professional motion graphics and visual effects software. It was used in this project to apply the rotoscoping technique, add visual effects, and combine all audio and video elements into the final output.

About this project

This 1-minute rotoscope animation combines original video footage, music, and text. Each frame was carefully traced and styled to create a smooth, stylized animation that tells a visual story.

What did I learn?

This project taught me how to use Adobe After Effects for frame-by-frame animation, how to sync audio with visuals, and how rotoscoping bridges the gap between live-action and animation.
